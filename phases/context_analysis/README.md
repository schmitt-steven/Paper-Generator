# Context Analysis Phase

Analyzes user code and notes to generate a paper concept.

## Components

- **`CodeAnalyzer`**: Analyzes code files for novel concepts and research relevance
- **`NotesAnalyzer`**: Extracts key findings, methodologies, and technical details from notes
- **`PaperConception`**: Generates paper concept with description, open questions, and code snippets

## Output

- `output/paper_concept.md` - Paper concept with description, open questions, and code snippets
